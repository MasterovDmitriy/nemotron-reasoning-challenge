# Nemotron Reasoning Challenge

Решение для соревнования [NVIDIA Nemotron Model Reasoning Challenge](https://www.kaggle.com/competitions/nvidia-nemotron-model-reasoning-challenge) на Kaggle.

Цель проекта — улучшить reasoning-точность моделей Nemotron на новом бенчмарке от NVIDIA, попутно построив переиспользуемый ML-пайплайн.

## Stack

- Python + uv (package management)
- Hydra (config management)
- DVC (data versioning)
- MLflow (experiment tracking)
- Pytest + GitHub Actions (testing & CI)
